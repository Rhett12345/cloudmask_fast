## 关键差异备注

1. **MERSI 无 3.959μm 通道**: MODIS 云检测核心通道, MERSI 最接近的是 ch20(3.8μm) 和 ch21(4.05μm)，两者与 MODIS 3.959μm 的 BT 差异在 +40K ~ +90K 量级
2. **MERSI ch20 和 ch21 白天饱和**: 动态范围上限分别为 350K 和 380K，白天晴空分别有 34.5% 和 17.9% 像素超出上限
3. **MERSI 无 6.75μm 通道**: 算法中用 ch22(7.2μm)+0.5K 代理，但对应代码已注释
4. **L1 HDF5 数据布局**:
   - `EV_1KM_Emissive[0:4]` = ch20, ch21, ch22, ch23
   - `EV_250_Aggr.1KM_Emissive[0:2]` = ch24, ch25
   - IR 标定: `Radiance = k0 + k1*DN + k2*DN² + k3*DN³` (IR_Cal_Coeff)
